PhoebY
A python library which provides web crawling capabilties. Good for web search engines. Uses breadth-first search.

About PhoebY

How to:
1) Change data/setting.phoeby
2) Run gather.py after importing it to your own python file, to crawl according to settings. In this example, test-gather.py does the crawling job
3) Import app.phoeby to your application
